## OpenPDF 3.0.1


* NEW TEXT COORDINATE EXTRACTION API
  - Integrated a new API for extracting precise text coordinates (PR #1442).
  - Enables developers to locate text positions for advanced document 
    analysis, automated form filling, and data scraping.

* CRITICAL BUG FIXES
  - Thai Glyph Rendering: Resolved IndexOutOfBoundsException in 
    FopGlyphProcessor during 1:N Thai Glyph Expansion. 
  - CSS Line-Height: Fixed incorrect handling of line-height CSS in 
    HTML -> PDF conversion for more accurate rendering.

* SECURITY & DEPENDENCIES
  - Bouncy Castle: Updated to 1.83 for latest cryptographic security.
  - Commons-IO: Bumped from 2.20.0 to 2.21.0.
  - Kotlin: Updated kotlin-maven-plugin and stdlib to 2.2.21.
  - HTML Parsing: Updated neko-htmlunit to 4.17.0.

* MAINTENANCE
  - Required Java version clarification added to README.md.
  - Test Framework: Upgraded JUnit version from 5.13.4 to 6.0.3.
  - Build System: Updated Maven plugins (Source, Javadoc, Release, 
    Checkstyle, and Surefire) to latest stable versions.
